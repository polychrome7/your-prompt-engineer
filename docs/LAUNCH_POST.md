# Launch Post

I built Your Prompt Engineer, a small agent skill for Codex and Claude Code.

It turns rough requests into dispatch-ready agent prompts, with:

- explorer/worker routing
- target validation
- confirmation before dispatch
- safety gates for risky tasks
- multilingual prompt handoff
- Codex and Claude Code prompt patterns

Useful when you know what you want, but not how to safely delegate it to an agent.

GitHub: https://github.com/polychrome7/your-prompt-engineer
